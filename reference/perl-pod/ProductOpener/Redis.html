NAME

ProductOpener::Redis - functions to push informations to redis

DESCRIPTION

ProductOpener::Redis is handling pushing info to Redis to communicate updates to openfoodfacts-search instance

$redis_client The connection to redis

init_redis($is_reconnect=0)

init $redis_client or re-init it if we where disconnected

it is uses ProductOpener::Config2::redis_url

push_to_search_service ($product_ref)

Inform openfoodfacts-search that a product was updated. It uses Redis to do that.

Arguments

Product Object $product_ref The product that was updated.
